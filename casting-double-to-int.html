Casting double to int

Friday, July 30th, 2017 04:08 - PST

If we go back in time, you were hypothetically creating a super awesome game. I had so generously helped you write your attack() method and you were very appreciative. You're welcome. Anyway, I wanted to talk about something that was in the attack() method that may not have made sense to you at the moment. I want to clear this up. Do you remember seeing...



That, my friend is called casting. The reason we have to do this in the attack() method is because Math.random() returns a double. However, the variable 'damage' is an int. In order to store the value in damage, we cast (Math.random() * ((max - min) + 1)) into an int by placing (int) before it. What this does, pretty much, is just remove the fractional part of the double. No it doesn not round it up. If the double is 14.99999999999, casting it to an int will make it 14.

WARNING:This cannot be done with a Double. Notice the capital 'D'. Although, Double does have a .intValue() method... but that's for a different post. Now get out there and start casting, you int mage.
