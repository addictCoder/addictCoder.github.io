Random Numbers with Math.random()

Friday, July 29th, 2017 03:07 - PST

Let's assume you're making a game and in that game, you've created many weapons for the player to use. Each weapon has a minimum and maximum damage value. Lets say it's the "Holy Club of Stupidity" with a +5 to rock throwing skill(HCoS). The HCoS has a minimum damage value of 5 and a maximum damage value of 20. How would we determine the amount of damage inflicted to an enemy character? Well, in case you didn't read the title of this post, Math.random().

What is Math.random()? First, divide it at the 'dot' operator into Math and .random(). Let's take a look at Math. Math is a class that is declared final. "public final class Math" This means Math is a class that cannot be subclassed. The other side, .random(), is a static method. Meaning this method cannot be overridden nor does the class need to be instantiated to call the method. So none of the "Math m = new Math(); m.random();". So what does all this do and how will this be good for the weapons of your awesome game?

The .random() method returns a double between 0.0(inclusive) and 1.0(exclusive). However, this doesn't cover the full range of weapons min and max damage values. So we need to create a formula to generate the random damage value between the min and max values. It will look something like this:

We want to cast that value to an int by putting (int) before the (Math.random() * 16). This takes the random value and removes the fractional part of the double.
